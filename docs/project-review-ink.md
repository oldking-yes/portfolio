## 项目真实内容

这是一个**数字水墨诗画互动平台**，不是笼统的"前端设计实验室"：

### 功能模块
1. **水墨场景引擎** — SVG + CSS Animations 构建动态背景
   - 山脉、竹林、仙鹤、锦鲤、墨滴、飞花瓣
2. **题诗作画** — Canvas 2D 绘画+题诗，一键导出
3. **五种主题配色** — 墨韵/青花/金碧/朱砂/雪景
4. **素笺留墨** — 书法风格留言墙
5. **琴韵音景** — Web Audio 生成古琴泛音

### 技术特点
- 零框架、零依赖、零构建
- 单个 index.html 自包含
- 全前端实现，无需后端
